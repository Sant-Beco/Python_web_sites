Introducción

Esta es la primera semana del curso CS50's Introduction to Programming with Python. Aquí aprendemos los conceptos más fundamentales: cómo crear funciones y trabajar con variables. Estos son los bloques de construcción de todo programa en Python.

Objetivos de aprendizaje:

  • Entender qué son las funciones y cómo crearlas
  • Trabajar con diferentes tipos de variables
  • Usar funciones built-in de Python como print(), input(), y len()
  • Comprender los tipos de datos: strings, integers, floats
  • Formatear strings con f-strings
⚙️

Funciones (Functions)

¿Qué es una función?

Una función es un bloque de código reutilizable que realiza una tarea específica. En Python, las funciones nos permiten organizar nuestro código y evitar repeticiones.

hello.py 
# Función básica sin parámetros
def hello():
    print("Hola, mundo!")
 
# Llamar a la función
hello()
 
# Función con parámetros
def saludar(nombre):
    print(f"Hola, {nombre}!")
 
saludar("Carlos")
saludar("María")

📝 Ejemplo Práctico: Mi Primera Función

mi_funcion.py 
def main():
    # La función main() es el punto de entrada de nuestro programa
    nombre = input("¿Cuál es tu nombre? ")
    saludar(nombre)
 
def saludar(nombre):
    print(f"Hola, {nombre}!")
 
# Ejecutar la función principal
if __name__ == "__main__":
    main()

Variables y Tipos de Datos

Variables en Python

Las variables son contenedores para almacenar valores. En Python, no necesitamos declarar el tipo de variable - Python lo infiere automáticamente (tipado dinámico).

🔤 Strings (str)

Texto entre comillas simples o dobles.

nombre = "Python"
apellido = 'Programming'
mensaje = """Texto
multilínea"""

🔢 Integers (int)

Números enteros sin decimales.

edad = 25
año = 2026
negativo = -42

💰 Floats (float)

Números con decimales.

precio = 19.99
pi = 3.14159
temperatura = -5.5
tipos_datos.py 
# Trabajando con diferentes tipos de datos
 
# Strings
nombre = "Ana"
apellido = "García"
nombre_completo = nombre + " " + apellido  # Concatenación
 
# Integers
edad = 28
año_nacimiento = 2026 - edad
 
# Floats
altura = 1.75
peso = 70.5
imc = peso / (altura ** 2)  # ** es exponenciación
 
# Conversión de tipos
edad_str = str(edad)  # 28 → "28"
numero = int("42")  # "42" → 42
decimal = float("3.14")  # "3.14" → 3.14
 
print(f"{nombre_completo} tiene {edad} años")

Formateo de Strings (f-strings)

F-Strings: La forma moderna de formatear

Los f-strings (formatted string literals) son la forma más legible y eficiente de formatear strings en Python 3.6+. Se crean anteponiendo una 'f' antes de las comillas.

fstrings.py 
# Forma antigua (evitar)
nombre = "Carlos"
edad = 25
print("Mi nombre es " + nombre + " y tengo " + str(edad) + " años")
 
# Con f-strings (recomendado) ✅
print(f"Mi nombre es {nombre} y tengo {edad} años")
 
# F-strings con expresiones
precio = 19.99
cantidad = 3
print(f"Total: ${precio * cantidad:.2f}")  # Total: $59.97
 
# F-strings multilínea
mensaje = f"""
Hola {nombre},
Tu edad es {edad} años.
El próximo año tendrás {edad + 1} años.
"""
print(mensaje)

Métodos de Strings

Métodos útiles para manipular strings

Python proporciona muchos métodos integrados para trabajar con strings. Estos nos permiten modificar, analizar y transformar texto fácilmente.

string_methods.py 
# Métodos de transformación
texto = "  Hello, World!  "
 
print(texto.strip())      # Elimina espacios al inicio y final
print(texto.lower())      # Convierte a minúsculas
print(texto.upper())      # Convierte a mayúsculas
print(texto.capitalize()) # Primera letra mayúscula
print(texto.title())      # Cada palabra con mayúscula inicial
 
# Métodos de búsqueda
frase = "Python es genial"
print(frase.startswith("Python"))  # True
print(frase.endswith("genial"))    # True
print(frase.find("es"))            # 7 (índice donde empieza)
print(frase.replace("genial", "increíble"))
 
# Dividir y unir strings
palabras = "Python,Java,JavaScript"
lista = palabras.split(",")  # ['Python', 'Java', 'JavaScript']
print(lista)
 
unido = " - ".join(lista)  # 'Python - Java - JavaScript'
print(unido)

Operadores Matemáticos

Operadores Básicos

  • + - Suma
  • - - Resta
  • * - Multiplicación
  • / - División (resultado float)
  • // - División entera
  • % - Módulo (residuo)
  • ** - Exponenciación

Orden de Operaciones

Python sigue el orden matemático estándar (PEMDAS):

  • Paréntesis ()
  • Exponentes **
  • Multiplicación/División *, /, //, %
  • Suma/Resta +, -
operadores.py 
# Operaciones básicas
x = 10
y = 3
 
print(x + y)   # 13
print(x - y)   # 7
print(x * y)   # 30
print(x / y)   # 3.3333...
print(x // y)  # 3 (división entera)
print(x % y)   # 1 (residuo)
print(x ** y)  # 1000 (10 elevado a 3)
 
# Orden de operaciones
resultado = 2 + 3 * 4  # 14, no 20
resultado2 = (2 + 3) * 4  # 20
 
# Operadores de asignación compuestos
contador = 5
contador += 1   # contador = contador + 1 → 6
contador *= 2   # contador = contador * 2 → 12
contador -= 3   # contador = contador - 3 → 9

Return Values - Funciones que Devuelven Valores

¿Qué es return?

Hasta ahora hemos usado funciones que hacen algo (imprimir en pantalla). Pero las funciones también pueden devolver valores que podemos usar en otras partes de nuestro código.

return_values.py 
# Función que devuelve un valor
def cuadrado(n):
    return n * n
 
# Usar el valor retornado
resultado = cuadrado(5)
print(resultado)  # 25
 
# Usar directamente en una operación
print(cuadrado(3) + cuadrado(4))  # 9 + 16 = 25
 
# Función con múltiples returns
def es_par(numero):
    if numero % 2 == 0:
        return True
    else:
        return False
 
print(es_par(4))  # True
print(es_par(7))  # False

📝 Ejemplo: Calculadora Simple

calculator.py 
def main():
    x = int(input("¿Cuál es x? "))
    y = int(input("¿Cuál es y? "))
    
    print(f"{x} + {y} = {sumar(x, y)}")
    print(f"{x} - {y} = {restar(x, y)}")
    print(f"{x} × {y} = {multiplicar(x, y)}")
    print(f"{x} ÷ {y} = {dividir(x, y):.2f}")
 
def sumar(a, b):
    return a + b
 
def restar(a, b):
    return a - b
 
def multiplicar(a, b):
    return a * b
 
def dividir(a, b):
    return a / b
 
if __name__ == "__main__":
    main()

Scope - Alcance de Variables

Variables Locales vs Globales

El "scope" determina dónde una variable puede ser accedida. Las variables definidas dentro de una función son locales (solo existen en esa función). Las variables fuera de funciones son globales.

scope.py 
# Variable global
nombre = "Python"
 
def saludar():
    # Variable local
    mensaje = "Hola"
    print(f"{mensaje}, {nombre}!")  # Puede acceder a 'nombre'
 
saludar()  # Funciona: "Hola, Python!"
# print(mensaje)  # Error: mensaje no existe aquí
 
# Mejor práctica: pasar variables como parámetros
def saludar_mejor(lenguaje):
    mensaje = "Hola"
    print(f"{mensaje}, {lenguaje}!")
 
saludar_mejor("Python")

⚠️ Buena Práctica

Es mejor pasar valores como parámetros en lugar de usar variables globales. Esto hace tu código más claro, más fácil de probar y menos propenso a errores.

5. Tip Calculator

Calcula propina y total de una cuenta.

tip.py 
def main():
    dollars = dollars_to_float(input("How much was the meal? "))
    percent = percent_to_float(input("What percentage would you like to tip? "))
    tip = dollars * percent
    print(f"Leave ${tip:.2f}")
 
def dollars_to_float(d):
    # Eliminar $ y convertir a float
    return float(d.replace("$", ""))
 
def percent_to_float(p):
    # Eliminar % y convertir a decimal
    return float(p.replace("%", "")) / 100
 
main()

Mis Notas y Aprendizajes

Lo que aprendí esta semana:

  • Las funciones hacen el código más organizado y reutilizable
  • Python tiene tipado dinámico - no necesito declarar tipos
  • Los f-strings son la forma moderna y mejor de formatear strings
  • La función main() es una convención importante en Python
  • Siempre debo validar la entrada del usuario

⚠️ Errores Comunes a Evitar

  • Olvidar los paréntesis () al llamar una función
  • No convertir tipos cuando es necesario (ej: input() siempre devuelve string)
  • Usar + para concatenar strings con números sin convertir
  • No usar f-strings y concatenar manualmente
