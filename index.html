¿Qué es Python?

Lenguaje de Alto Nivel

Python es un lenguaje de programación interpretado, de alto nivel y de propósito general. Su filosofía de diseño enfatiza la legibilidad del código con el uso de espacios en blanco significativos.

Multipropósito

Desde desarrollo web hasta inteligencia artificial, análisis de datos, automatización, ciencia de datos y más. Python es versátil y poderoso.

Sintaxis Clara

Python se lee casi como inglés (o español). Su sintaxis limpia permite que los principiantes aprendan rápidamente mientras mantiene la potencia para proyectos complejos.

Comunidad Global

Una de las comunidades de desarrolladores más grandes del mundo. Miles de librerías, frameworks y recursos de aprendizaje disponibles.

02

Historia y Orígenes

1989

Los Inicios

Guido van Rossum comienza a trabajar en Python durante las vacaciones de Navidad en el CWI (Centrum Wiskunde & Informatica) en Amsterdam, Países Bajos.

1991

Primera Versión Pública

Python 0.9.0 es lanzado. Ya incluía clases con herencia, manejo de excepciones, funciones y los tipos de datos principales.

2000

Python 2.0

Introducción de list comprehensions, garbage collection y soporte para Unicode. Comienza la era moderna de Python.

2008

Python 3.0

Lanzamiento de Python 3, una revisión mayor del lenguaje. No es completamente compatible con Python 2, pero introduce mejoras fundamentales.

2020

Fin de Python 2

Python 2.7 alcanza su fin de vida (EOL). La comunidad se consolida completamente en Python 3.

2026

Mi Viaje Comienza

Inicio mi aprendizaje estructurado de Python con CS50, documentando cada paso, desde funciones y variables hasta programación orientada a objetos.

🐍 ¿Por qué "Python"?

Guido van Rossum era fan del grupo de comedia británico "Monty Python's Flying Circus". Cuando necesitó un nombre para su nuevo lenguaje, quería algo corto, único y ligeramente misterioso, así que eligió "Python". No tiene nada que ver con la serpiente, aunque el logo sí las incluye.

03

Características Principales

🔤 Tipado Dinámico

No necesitas declarar tipos de variables. Python infiere el tipo automáticamente, lo que hace el código más conciso y flexible.

🎯 Interpretado

No requiere compilación. El código se ejecuta directamente línea por línea, facilitando el testing y debugging.

📦 Baterías Incluidas

Python viene con una biblioteca estándar extensa que cubre todo desde matemáticas hasta networking y procesamiento de archivos.

🌐 Multiplataforma

Funciona en Windows, macOS, Linux y más. Escribe una vez, ejecuta en cualquier lugar.

🧩 Orientado a Objetos

Soporta programación orientada a objetos, pero también paradigmas procedurales y funcionales.

🔓 Open Source

Completamente gratuito y de código abierto. Una comunidad global contribuye constantemente a su mejora.
